# 화면 설계 — 나만의고물상

> 간단함이 정체성. 「금 모으기」 레이아웃 참고. 딸기 idle 화면 자산 재사용.

## 화면 목록
1. **메인(고물상)** — 핵심
2. **창고/판매** — 창고 고물 판매 + 엽전 → 토스 포인트 교환
3. **출석** — 광고 보고 1원 (하루 5회)
4. **안내** — 고물 종류/시세, 모으는 법, 교환 규칙, 정책/고객센터
5. **인트로** — 첫 실행 1회

## 1) 메인(고물상) 화면 — 금모으기 레이아웃 참고
```
┌─────────────────────────────┐
│ (토스 호스트 헤더: 앱이름 + ✕)  │
├─────────────────────────────┤
│   오늘의 고철 시세            │
│   ⚙️ 1kg  3,200원   (시세)    │  ← 금모으기 '금 1돈 값' 자리
│                               │
│   ┌───────────────────────┐   │
│   │ 최대 적재량  120개        │   │
│   │ 수집 속도   100%         │   │
│   │     🛒 손수레 (고물 쌓임)  │   │  ← idle 적재 시각화
│   │   [📺 추가로 줍기 (광고)]   │   │
│   └───────────────────────┘   │
│                               │
│   지금까지 모은 엽전           │
│   0 0 1 2 4 0  냥  (오도미터)  │  ← 굴러가는 카운터
│                               │
│   [⚡ 빠르게 모으기(광고)]      │
│   [📦 창고로 옮기기(광고)]      │
│                               │
│   📅 일주일 개근 +10%          │
│   [🎁 광고 보고 1원 · 5회]     │
│   [🏬 창고 / 판매·교환 →]      │
├─────────────────────────────┤
│ [📢 배너 광고]                 │
└─────────────────────────────┘
```
- 들어오면 **방치 누적 수확 팝업** ("자는 동안 고물 34개 주웠어요!").
- 손수레 탭 → 소소한 고물(체류감).

## 2) 창고/판매 화면
- 창고 고물 종류별 개수 + 시세 → "전부 판매" → 엽전 획득.
- 엽전 잔액 → "토스 포인트로 받기"(광고) → 고정환율 지급. 일일/누적 캡 안내.
- 용어: **"엽전"**, **"토스 포인트로 받기"** (출금/인출 X).

## 3) 안내 화면
- 고물 4종·시세, 방치 vs 활동 차이(광고 보면 비싼 고물↑), 엽전·교환·캡 안내.
- "꽝 없어요" / 도박성 표현 없음 / "뽑기·룰렛" 단어 금지. 개인정보·고객센터.

## UX 라이팅
- 해요체, B급/정겨운 고물상 감성. "고물도 모으면 돈이 돼요!"
- "포인트"(인앱) X, "출금" X.

## 견고성
- `allowFontScaling=false`, 가로 카드 `numberOfLines+adjustsFontSizeToFit`, 인라인 SVG(이미지 캐시 무관).
