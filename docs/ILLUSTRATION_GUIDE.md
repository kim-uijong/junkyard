# 일러스트 가이드 — 나만의고물상

> 인라인 SVG 우선(딸기 코드처럼) → 외부 이미지 없음 = 캐시 문제 원천 차단. 자산 최소화.

## 스타일
- 3등신 SD/플랫 + 부드러운 그림자. **B급·레트로 고물상** 감성(정겨움 + 약간 코믹).
- 컬러: 녹슨 금속/크라프트 갈색 + 포인트 컬러. 엽전은 황동색.

## 필요 자산 (SVG로 직접)
| 구분 | 자산 |
|---|---|
| 🛒 손수레 | 빈 손수레 → 고물 조금 → 수북이(적재량 3~4단계) |
| 고물 4종 아이콘 | 📄 폐지 / 🍾 공병 / ⚙️ 고철 / 🔩 특수(구리) |
| 🏬 창고 | 고물 쌓인 창고/적치장 |
| 🪙 엽전 | 옛 동전(가운데 네모 구멍) — 재화 아이콘 |
| UI 아이콘 | 줍기/부스터(자석크레인)/판매/교환 |
| 앱 아이콘·썸네일 | 1 + 1 |

→ 딸기 SVG 컴포넌트 구조(스테이지별 G) 그대로 차용. 손수레 적재 단계 = 딸기 성장 단계 매핑.

## 제작
- 인라인 SVG(react-native-svg) 직접 작성이 1순위.
- 보조로 ChatGPT/Gemini(썸네일·아이콘). 호스팅 필요 시 GitHub raw + `?v=${Date.now()}`.

## 모티프
- 손수레 = 금모으기의 '수레'. 창고 = '금고'. 자석크레인 = '빠르게 모으기'.
- 엽전(상평통보 느낌) — 토스 포인트와 시각적으로 확실히 구분.
